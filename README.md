# Texas Schools Performance Analysis

##  AI Acknowledgment
**Note:** This project, including all code logic, data processing pipelines, and documentation, was developed and co-authored with the assistance of **Google Gemini AI**.

## Project Description
This project focuses on predicting Texas school accountability ratings using data from Data.gov. We implemented an end-to-end pipeline including data preprocessing, feature engineering, classification (Random Forest/XGBoost), and unsupervised clustering.

## Repository Structure
- `/data`: Source datasets.
- `/notebooks`: Jupyter Notebooks with full analysis.
- `/figures`: Key visualizations and model outputs.
